Holistic Health Management

Discover a unique blend of physical and spiritual health management at Bhakti Hospital. Our state-of-the-art system leverages the power of JavaScript, Express.js, and MongoDB to deliver exceptional care and insights into your well-being.

Features

  • Comprehensive user health management
  • Real-time health updates
  • Spiritual and physical health integration
  • Secure authentication and data validation

Available Routes

Route Method Description
/ GET Welcome route that sends a welcome message.
/getUser GET Retrieves user data based on the provided id query parameter.
/healthReport GET Generates a health report for a user based on the provided id query parameter.
/updateOrgan PUT Updates the health status of an organ for a user. Requires id, organ, and ishealthy in the request body.
/deleteOrgan DELETE Removes all unhealthy organs for a user based on the provided id query parameter.
/addOrgan POST Adds a new organ (heart or kidney) to a user. Requires id, organ, and ishealthy in the request body.
/auth/signin POST Allows users to sign in. Validates user information, creates a JWT token, and saves the user to the MongoDB database.
/auth/users GET Retrieves user information for authenticated users. Uses JWT token for authentication.
